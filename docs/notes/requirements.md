# Campus Management Software

## Target
Everybody on campus

## Description
Everybody can reserve rooms

## Requirements
* Accounts
    * Everybody can register
    * Stay logged in
    * Administrator
        * Change all reservations
        * Change rooms
        * *Can assign other administrators*
    * Normal users
        * Reserve rooms
        * Cancel reservations
* Reservations
    * Fixed interval timeslots
* Interface
    * Information about buildings/rooms
    * Available timeslots
    * Booked rooms
